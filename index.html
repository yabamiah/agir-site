AGIR - Automação para uma Governança Inteligente e Responsável no DF

Sobre o Projeto

O projeto AGIR tem como objetivo analisar a disseminação da governança pública e dos programas de integridade no setor público do Distrito Federal. Entre as principais metas estão:

  • Desenvolver um sistema de automação (RPA) para buscar programas de integridade.
  • Criar um painel de transparência ativa para verificação da conformidade.
  • Elaborar um indicador de conformidade dos programas de integridade.

Metodologia

Para atingir seus objetivos, o projeto utilizará tecnologias avançadas para coleta de dados automatizada e análise de governança pública, além de realizar estudos aprofundados sobre melhores práticas de integridade no setor público.
